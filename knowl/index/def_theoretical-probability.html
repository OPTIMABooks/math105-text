Definition 3.5.2.

The theoretical probability of a certain event \(E\) happening in an experiment is found by counting the number of possible outcomes of the experiment in which \(E\) happens and dividing by the total number of outcomes, assuming each outcome is equally likely. That is:
\begin{equation*} P(E)=\frac{\text{number of outcomes corresponding to the event E}}{\text{total number of equally-likely outcomes}} \end{equation*}
