Definition 4.2.31.
A stem-and-leaf plot is a table where each data value is split into two parts: the stem (all but the right-most digit) and the leaf (the right-most digit). The table includes a row for each stem, with the stem in the first column and a list of its corresponding leaves in the second column, arranged in ascending order without spaces or commas.
