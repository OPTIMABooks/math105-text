Example 2.4.8.

Evaluate the validity of the following argument.
🔗
If taxes are cut, then retail sales increase. If retail sales increase, then the economy improves. Thus, if taxes are not cut, the economy will not improve.
🔗
🔗
Solution.
First, define the necessary logical variables:
  • Let \(p\) = “Taxes are cut.”
    🔗
  • Let \(q\) = “Retail sales increase.”
    🔗
  • Let \(r\) = “The economy improves.”
    🔗
The argument is summarized symbolically as
🔗
P1: \(p \to q\text{,}\)
P2: \(q \rightarrow r\text{,}\)
C: \(\lnot p \rightarrow \lnot r\text{.}\)
Next, create a truth table for the above argument. Make sure each premise and the conclusion appear in a column. Here we number the rows for easy reference, but in general this is not necessary.
🔗
row \(p\) \(q\) \(r\) P1: \(p\to q\) P2: \(q\to r\) \(\neg p\) \(\neg r\) C: \(\neg p \to \neg r\)
1 T T T T T F F T
2 T T F T F F T T
3 T F T F T F F T
4 T F F F T F T T
5 F T T T T T F F
6 F T F T F T T T
7 F F T T T T F F
8 F F F T T T T T
Notice that in rows 1 and 8 each premise and the conclusion are true. This might make you think the argument is valid. However, in rows 5 and 7 the premises are true but the conclusion is false! This demonstrates that the argument is invalid. To be valid every row where the premises are true must have a true conclusion as well.
