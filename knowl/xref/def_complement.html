Definition 3.1.12.

The complement of a set \(A\) in a universal set \(U\) is the set \(A^C\) consisting of all elements in \(U\) that are not in \(A\text{.}\) That is,
\begin{equation*} A^{C} = \{x\in U\mid x \text{is NOT in } B \}. \end{equation*}
