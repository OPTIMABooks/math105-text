    Section 4.3 Measures of Center

    Objectives

    • Describe the ‚Äúcenter‚Äù of a data set using the mean, mode, and median, and describe the strengths and weaknesses of each of these measures.
      üîó
      üîó
    üîó
    In addition to graphical and verbal descriptions, we can use numerical summaries to better understand quantitative distributions by examining where the data is centered and how spread out the values are. Together, measures of center and spread provide important information for estimating population parameters. This section focuses on measures of center, while the next will address measures of spread. We will begin by exploring three common ways to describe the center of a distribution: the mode, median, and mean, each suited to different situations.
    üîó

    Subsection 4.3.1 Mode

    We have already seen the term mode used in the context of a graph. Recall that the mode of a histogram or bar graph is the class or category that has the tallest bar. That is, it is the class or category that appears the most in the data set. This seems to be a reasonable way to describe a ‚Äútypical‚Äù element of a data set and leads to the following definition.
    üîó

    Definition 4.3.1.

    The mode of a set of values is the value that appears most often. If more than one value is tied for the most appearances, each one is a mode. If no value appears more than once, there is no mode.
    üîó
    üîó
    The mode is unique among our measures of center as it is the only one that applies to both quantitative variables and qualitative variables. Consider the following examples.
    üîó

    Example 4.3.2.

    A breakout presentation at a business convention attracts 16 participants. The job title of each of these participants is as follows. Find the mode of this set of data.
    üîó
    Solution.
    The mode is the most commonly appearing value. One way to find that is to construct a frequency table. Doing so for this data produces the following table, from which we can see that the mode is ‚Äúmanager‚Äù.
    üîó
    üîó
    üîó

    Example 4.3.5.

    A survey of twelve grocery store shoppers asked how many times a month these shoppers visit their favorite grocery store. Find the mode of the resulting data, shown below.
    üîó
    Solution.
    Again, the best starting point is to construct a quick frequency table.
    üîó
    Notice that in this data set, both the 5 and the 3 appear three times. Every other value appears two or fewer times. This makes this set of data bimodal with modes 3 and 5.
    üîó
    üîó
    üîó
    While the mode is an easy way to measure the center, and is in fact the only way we have to measure the center of a set of qualitative data, it is usually not the best choice for measuring the center of a set of quantitative data. To see why this may be so, consider the next example.
    üîó

    Example 4.3.8.

    Find the mode of the following sets of numbers.
    1. \(\displaystyle \lbrace 1, 2, 5, 7, 12, 15, 19, 22, 50 \rbrace\)
      üîó
      üîó
    2. \(\displaystyle \lbrace 1, 1, 20, 23, 26, 24, 29, 30, 27, 32, 19\rbrace\)
      üîó
      üîó
    üîó
    Solution.
    The modes of these sets of numbers are as follows:
    1. This data set contains no repeated values. Therefore, according to the definition, there is no mode for this set.
      üîó
      üîó
    2. This data contains only one repeated value, the 1. So the mode is 1. However, 1 is definitely not the ‚Äútypical‚Äù value in the set since all other values are between 19 and 32.
      üîó
      üîó
    üîó
    üîó
    üîó
    In summary, you should use the mode to measure the center of any qualitative set of data, or as a quick, but not definitive, measure of center for quantitative data. But be careful! The mode may not exist, there may be multiple modes, and if there is a single mode it may be very different from the ‚Äútypical‚Äù values in the data set.
    üîó
    üîó

    Subsection 4.3.2 Midrange

    When we think of the ‚Äútypical‚Äù value in a set of data, we may also think of the word middle. Our next measure of center, the midrange, is exactly that‚Äîthe middle of the data set. Note that the midrange is not a very useful measure of center. Rather, we introduce it in order to discuss outliers and why some measures of center are more appropriate than others depending on characteristics of the data.
    üîó

    Definition 4.3.9.

    The midrange of a set of data is the value in the exact center of the data. To compute the midrange, use the formula:
    \begin{equation*} \text{midrange} = \frac{\text{maximum} + \text{minimum}}{2} \end{equation*}
    where maximum is the maximum value in the data set and minimum is the minimum value.
    üîó
    üîó

    Example 4.3.10.

    A certain pond has been well stocked with fish for the upcoming fishing season. In order to keep a fish from this pond, it must be at least 13 inches long. To try to determine if the fish in the pond are long enough to allow fishing, a fish and game warden randomly catches and releases 20 different fish, finding the following lengths, in inches. Find the midrange of these data.
    üîó
    Solution.
    To find the midrange we need to identify the maximum and minimum values in the data set. The largest value is 18.3 and the smallest is 6.5, Thus we compute the midrange using:
    \begin{equation*} \text{midrange} = \frac{18.3+6.5}{2} = 12.4\text{.} \end{equation*}
    It looks like the pond is not yet ready to open for fishing. The typical fish, as measured by the midrange, is only 12.4 inches long and could not be kept.
    üîó
    üîó
    üîó
    The midrange is useful for measuring the center of a dataset, but it has significant limitations. Most importantly, the midrange is extremely sensitive to outliers. Additionally, it only considers the maximum and minimum values, ignoring all other data points. Ideally, a measure of center should incorporate every value in the dataset when determining the ‚Äútypical‚Äù value.
    üîó

    Definition 4.3.12.

    An outlier in a data set is a value that is either much larger than the rest of the data or much smaller than the rest of the data.
    üîó
    üîó
    To see how an outlier can affect the midrange, let‚Äôs take another look at the fish-length data from the previous example.
    üîó

    Example 4.3.13.

    The game warden believes that the 6.5 inch fish she caught in the sample above is an outlier. She decides to redo the computation without that fish. Find the new midrange.
    üîó
    Solution.
    Without the 6.5 inch outlier, the new minimum is 8.7. This gives the following values:
    \begin{equation*} \text{midrange} = \frac{18.3+8.7}{2} = 13.5\text{.} \end{equation*}
    With this new computation, it looks like the fish are in fact large enough to be caught‚Äîthe typical fish length, as measured by the midrange, is 13.5 inches.
    üîó
    üîó
    üîó
    Notice what a big difference the single 6.5 inch fish made! This is what we mean when we say that the midrange is sensitive to outliers.
    üîó
    üîó

    Subsection 4.3.3 Median

    To reinforce why we need a measure of center that is resistant to outliers, consider the following example.
    üîó

    Example 4.3.14.

    The most expensive house in a certain market is a $2,500,000.00 mansion. The least expensive house is an $8,000.00 mobile home. Do you think that the midrange is an accurate measure of the ‚Äútypical‚Äù home value in this market?
    üîó
    To answer this question, let‚Äôs first compute the midrange. This is:
    \begin{equation*} \text{midrange} = \frac{2500000 + 8000}{2} = 1254000. \end{equation*}
    Do you think that the typical house really costs one and a quarter million dollars? Probably not! The midrange is not an appropriate measure of center for this data.
    üîó
    üîó
    As we saw before, the outliers in this data set are affecting the midrange. To address these limitations, we need a measure of center that is resistant to outliers and uses more than just the extreme values.
    üîó

    Definition 4.3.15.

    The median of a data set is the middle number, or average of the two middle numbers, in the set of data once it has been arranged in increasing order.
    üîó
    üîó
    You may have heard of the median, a common measure of center that‚Äôs especially useful when a data set includes outliers, such as housing prices or household incomes. The term ‚Äúmedian household income‚Äù is often used because it reflects typical earnings more accurately in such cases. This is especially helpful in places like Washington State, where billionaires like Bill Gates can skew the data. Let‚Äôs practice finding the median.
    üîó

    Example 4.3.16.

    Find the median of each set of data below.
    1. \(\displaystyle \lbrace 14, 17, 10, 9, 16, 19, 12, 10, 15, 13 \rbrace\)
      üîó
      üîó
    2. \(\displaystyle \lbrace 100, 215, 164, 117, 1000, 205, 178 \rbrace\)
      üîó
      üîó
    üîó
    Solution.
    To find the median, first arrange the data values from least to greatest, then locate the middle value. Let‚Äôs apply this process to both data sets above.
    1. Our first step is to re-arrange this set of numbers in order, from smallest to largest:
      \begin{equation*} \lbrace \overbrace{9, 10, 10, 12,}^{\text{lower half}} \overbrace{13, 14,}^{\text{middle pair}} \overbrace{15, 16, 17, 19}^{\text{upper half}}\rbrace \end{equation*}
      Then we find the middle number. Since there are an even number of values (ten of them), the exact middle lies half way between the two center values (5th and 6th) which are 13 and 14. So the median is \(\frac{13+14}{2} = 13.5\text{.}\)
      üîó
      üîó
    2. Again, we start by ordering the data:
      \begin{equation*} \lbrace \overbrace{100, 117, 164,}^{\text{lower half}} \overbrace{178,}^{\text{median}} \overbrace{205, 215, 1000}^{\text{upper half}} \rbrace \end{equation*}
      As there are an odd number of values in this data set (seven of them), the median is the single value in the exact center (the 4th value), which is 178.
      üîó
      üîó
    üîó
    üîó
    üîó
    Notice that the 1000 in the second data set is an outlier. But when finding the median, the actual value doesn‚Äôt matter‚Äîonly its position in the data set does. You could replace 1000 with a more typical number like 220, and the median would stay the same. Bill Gates could earn just $200,000 a year, and the median household income in Washington State wouldn‚Äôt change! This is what it means for the median to be resistant to outliers.
    üîó
    üîó

    Subsection 4.3.4 Mean

    While the median effectively handles outliers, it has one significant limitation: it ignores the values of individual data points. When we want each value to influence our measure of center, we use the arithmetic mean (commonly called the average). Unlike the median, the arithmetic mean incorporates every data point in the dataset. For a sample, we use \(\overline{x}\) (read ‚Äúx bar‚Äù) to represent the mean.
    üîó

    Definition 4.3.17.

    The arithmetic mean, which we will call simply the mean, is a measure of center found by taking the sum of all values in the data set and dividing by the number of values in the data set. Symbolically,
    \begin{equation*} \bar{x}=\frac{\text{sum of data values}}{n}\text{,} \end{equation*}
    where \(n\) is the number of values in the data set.
    üîó
    üîó
    The term arithmetic mean is important for two reasons. First, we avoid calling it the average because ‚Äúaverage‚Äù simply means ‚Äútypical,‚Äù and there are multiple ways to measure what‚Äôs typical. Second, we specify arithmetic mean because other types of means exist, though we don‚Äôt cover them here.
    üîó

    Example 4.3.18.

    Mirabel‚Äôs exam scores for her last math class were: 79, 86, 82, and 94. What is her mean test score?
    üîó
    Solution.
    To find the mean test score we need to find the sum of her test scores, then divide the sum by the number of test scores (\(n=4\)). The mean is:
    üîó
    \begin{equation*} \bar{x}=\frac{79+86+82+94}{4}=85.25\text{ points} \end{equation*}
    üîó
    Notice that the mean has the same units as the data and that it is important to label it.
    üîó
    üîó
    üîó
    It is reasonable to calculate the mean by hand when working with a small data set. However, if the data set is large or if you plan to compute additional statistics, using technology is a more efficient choice. Ask your instructor how to approach these types of problems. That said, when a large data set with repeated values is organized in a frequency table, it can still be practical to perform the calculation by hand as shown in the next example.
    üîó

    Example 4.3.19.

    One hundred families from a particular neighborhood are randomly selected and asked to give their annual household income rounded to the nearest $5,000. The results are shown in the frequency table below. Find the mean annual income for this neighborhood.
    üîó
    Income
    (thousands of dollars)    		 Frequency
    $15 6
    $20 8
    $25 11
    $30 17
    $35 19
    $40 20
    $45 12
    $50 7
    Solution.
    Calculating the mean by hand could get tedious if we try to type in all 100 values:
    \begin{equation*} \bar{x}=\frac{\overbrace{15+ \cdot + 15}^{6 \text{ terms}} + \overbrace{20+ \cdot + 20}^{8 \text{ terms}} +\cdots+ \overbrace{50 + \cdot + 50}^{7 \text{ terms}}}{100} \end{equation*}
    üîó
    However, we can calculate this easily using multiplication to deal with repeated values. Doing so, the sum of the data values (the numerator of the mean) is
    \begin{align*} \amp (15)(6)+(20)(8) + (25)(11)+(30)(17)\\ \amp +(35)(19)+(40)(20) +(45)(12)+(50)(7) \\ \amp= 3390 \text{.} \end{align*}
    The mean is then
    \begin{equation*} \bar{x}=\frac{3390}{100}=33.9 \end{equation*}
    That is, the mean household income of our sample is 33.9 thousand dollars or $33,900.
    üîó
    üîó
    üîó
    üîó

    Subsection 4.3.5 Which Measure Should I Use?

    We have now seen four different numerical measures of center:
    • the mode,
      üîó
      üîó
    • the midrange,
      üîó
      üîó
    • the median, and
      üîó
      üîó
    • the mean.
      üîó
      üîó
    How do we know when to use a particular measure of center? To better answer this question, let‚Äôs review the strengths and weaknesses of each of these numerical summaries.
    üîó
    As we can see, each method has its own strengths and weaknesses. Based on these, we summarize when to use which measures of center below.
    üîó

    Observation 4.3.21.

    • Mode.
      Use the mode when dealing with qualitative data, or for a quick measure of center for small sets of quantitative data. The mode may not exist, and should not be used in general for larger sets of quantitative data.
      üîó
      üîó
    • Midrange.
      The midrange may be used for a quick estimate of center and variation, but should not be used if there are any indications of outliers.
      üîó
      üîó
    • Median.
      Use the median if there are indications of strong outliers in your data. Data sets with strong outliers would include income or housing price data.
      üîó
      üîó
    • Mean.
      The mean is the best all-purpose measure of center and variation for quantitative data. Unless your data set has strong outliers, this is the summary value you should use.
      üîó
      üîó
    üîó
    üîó

    Example 4.3.22.

    Determine which measure of center would be best in each of the following data sets.
    1. The salaries of all employees at IBM
      üîó
      üîó
    2. A quick assessment of how well a group of athletes did in a 100-meter dash
      üîó
      üîó
    3. Determine the typical hair color in a group of teenagers.
      üîó
      üîó
    4. Finding the typical height of a pine tree in a given acre of forest.
      üîó
      üîó
    üîó
    Solution.
    1. Salaries of all employees at a large firm such as IBM are likely to include outliers, such as the CEO‚Äôs salary. For this reason, the median is probably the best measure of center.
      üîó
      üîó
    2. The midrange may be best. This set of data should not include many outliers, and the values will be small and relatively close together. This makes the midrange an acceptable way to quickly measure the center.
      üîó
      üîó
    3. Hair color is a qualitative variable, so our only choice is to use the mode.
      üîó
      üîó
    4. Pine tree heights on an acre of forest land should not have large outliers, and there are likely to be quite a few trees. Since there is no reason to use any of the other measures of center, the mean would be best here.
      üîó
      üîó
    üîó
    üîó
    üîó
    It is often helpful to compute multiple summary statistics for comparison. One useful comparison is between the mean and the median. If the mean is greater than the median, the data distribution is typically skewed right. If the mean is less than the median, the distribution is skewed left. When the mean and median are approximately equal, the distribution is roughly symmetric.
    üîó

    Example 4.3.23.

    Recent college graduates were asked how much student loan debt they have. The data has a mean of $46,265 and a median of $33,652. Based on this information, do you expect the distribution to be symmetric, skewed to the left, or skewed to the right?
    üîó
    Solution.
    Since the mean is greater than the median, we expect the distribution to be skewed to the right.
